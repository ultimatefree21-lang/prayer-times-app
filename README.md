# Prayer Times App - تطبيق مواقيت الصلاة

تطبيق أندرويد حديث لعرض مواقيت الصلاة مع الموقع الجغرافي والمزايا التالية:

## المزايا:
✅ عرض مواقيت الصلاة (الفجر، الظهر، العصر، المغرب، العشاء)
✅ تحديد الموقع الجغرافي تلقائياً
✅ قاعدة بيانات محلية للمواقيت
✅ واجهة مستخدم حديثة بـ Jetpack Compose
✅ دعم اللغة العربية

## البنية المعمارية:
- **MainActivity**: الواجهة الرئيسية
- **LocationService**: خدمة تحديد الموقع
- **PrayerTimeDao**: عمليات قاعدة البيانات
- **PrayerTimeViewModel**: إدارة البيانات
- **PrayerTimeDatabase**: قاعدة البيانات

## التكنولوجيات المستخدمة:
- Kotlin
- Jetpack Compose
- Room Database
- Google Location Services
- Coroutines

## التثبيت والتشغيل:
1. استنساخ المستودع
2. فتح المشروع في Android Studio
3. تشغيل التطبيق على جهاز أو محاكي
